Fonterra Farm Source was the first major project I was involved on during my time at Solnet. The project was a complete rebuild of Fonterra online farmer portal (previously called Fencepost). This has been a long running project, I was involved from the very as a part small team working on the first pieces of core functionality. I worked mostly with the HTML/CSS and javascript code: implimenting

The front end of Farm Source is built on SilverStripe CMS. To achieve the required level of interactivity, we integrated Angular JS with SilverStripe. At the time this was an innovative combination of technologies, and took some The back end is a Java application the uses ETL to ingest data from a variety of Fonterra's core data systems. It provides access to the data via a rest api, which is also used by the Farm Source mobile app.
