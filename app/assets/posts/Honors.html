The goal of my project was to create a tool for collaborative distributed planning on software development projects. The system I created utilised Anoto digital pens; to  support and enhance the tried and tested agile planning technique of using stories and index cards. 

Anoto pens:

The Anoto pen looks and feels like a traditional ball point pen but, it is amalgamated with a digital camera, an advanced image microprocessor, and a Bluetooth transmitter. When a user writes on paper augmented with the Anoto dot pattern the camera in the pen reads these dots, and uses the pattern to uniquely identify the pens exact location. The location of the pen is streamed via Bluetooth so that computer software can receive the information and process it to create a digital copy of the markings on the Anoto paper.
